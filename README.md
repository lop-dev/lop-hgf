# 完美联盟（Lop : League of perfect）
#### 我们是一帮专注于游戏圈的程序员，凭借多年的项目经验，历时十多年的时间，开发和维护了这套前后端框架；
#### 开发这套底层和框架的目标：减少重复劳动、高效稳定、快速启动新的项目、将已有项目中不稳定部分替换掉；
#### 高效稳定的底层和框架，会大大节约开发成本、缩短开发周期、提升开发效率、增强成员信心、提高成功概率。

## 技术保障
#### 本套底层框架以及实例库，同时适用游戏软件开发和应用软件开发，大家可以下载试用；
#### 目前有多款线上游戏使用本套框架，稳定、并发、性能都得到验证，大家可以放心使用。
